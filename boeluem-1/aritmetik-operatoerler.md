# Aritmetik Operatörler

Aritmetik operatörler programlamada matematiksel işlemler yapabilmemize olanak sağlar.

| Operatör | Açıklama | Örnek |
| :--- | :--- | :--- |
| + | Toplar | 2+5 |
| - | Çıkarır | 10-3 |
| \* | Çarpar | 3\*4 |
| / | Böler | 10/2 |
| % | Bölümden kalanı verir \(Mod\) | 10%3 |
| ++ | 1 arttırır | 1++ |
| -- | 1 eksiltir | 1-- |
